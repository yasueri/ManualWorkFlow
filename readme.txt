■構成メモ
WorkFlow/
│── index.html              # トップページのHTMLファイル。CSS・JavaScriptはhtmlファイルにまとめています
│── data/                   # メインコンテンツに流しこむCSVデータを貼り付けるjs置き場
│   └── lt-csv.js           # 暫定ファイル名
│── css/                    # CSSファイル
│   └──style.css            # メイン
│── js/                     # JavaScriptファイル
│   ├──script.js            # メイン
│   └──keyboard-manager.js  # キー操作部分
└── pages/                  # メインコンテンツ
    └── lt.html             # 暫定ファイル名


■主な仕様
・CSVデータをもとに表示内容を更新
　　htmlファイルの該当部データを差し替えるだけです
・選択やり直し機能
　　以前の選択肢からやり直せます
・同一選択肢の自動選択機能
　　同一選択肢が表示されると、前回と同一の選択肢が自動的に選択されます
　　矛盾が生じないよう、この場合は選択肢を変更できないようにしています
・指定文字を赤色で強調表示
　　現在の対象文字：「要印刷」「画面」
・UIに配慮
　　・スタート地点を選択できる（ハンバーガーメニュー）
　　・トップページはレスポンシブ対応
・セキュリティに配慮
　　外部テキスト入力の欠如
　　　　ユーザーからテキスト入力を受け付けておらず、すべてのデータは事前定義されています。
　　閉じた環境
　　　　アプリケーションは外部リソースにアクセスせず、APIを使用せず、単独で動作します。
　　静的コンテンツ
　　　　CSVデータ自体に悪意のあるコードが含まれる場合はリスクとなり得ますが、
　　　　これは開発者が制御できるものであり、実質的なリスクとはみなしていません。
　　　　念のため、基本的なサニタイズ処理を施しています。
・キー操作
　　キー入力でも操作可
　　　　pagesのhtmlファイルに操作ガイドを表示
