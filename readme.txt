■構成メモ

│── index.html         # トップページのHTMLファイル。CSS・JavaScript：htmlファイルに含む

└── assets/            # 画像・マニュアル・その他のアセット

    ├── videos/        # 動画ファイル
    └── manuals/       # マニュアル ※マニュアルへのリンクを設定する場合はここに格納
